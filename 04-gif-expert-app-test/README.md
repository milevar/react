# GifExpertApp

Una pequeña aplicación hecha en React con Hooks para buscar y mostrar gifs animados.

### http-server: a simple static HTTP server

```
npm install --global http-server
```

https://www.npmjs.com/package/http-server

### Build and Test

```
npm run build
cd build
http-server -o
```
